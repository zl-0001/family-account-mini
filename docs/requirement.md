# 家庭记账需求说明

## 一、产品定位

面向家庭用户的智能记账工具，支持多成员协作、收支分类管理、预算控制和投资理财追踪。

### 2.2 自定义分类

用户可创建自定义支出分类，支持：
- 设置分类名称、图标、颜色
- 选择所属大类（孩子支出/家庭固定/家庭不固定/旅行/成长/其他）
- 设置是否计入预算
- 排序优先级

### 2.3 分类管理功能

- 查看所有分类及使用统计
- 编辑分类信息（名称、图标、颜色）
- 删除分类（需确认无关联记录）
- 禁用/启用分类
- 分类使用排行榜

### 4.2 投资记录功能

#### 4.2.1 买入记录

| 字段 | 说明 |
|------|------|
| 投资名称 | 如：沪深300ETF、招商银行理财 |
| 投资类型 | 股票/基金/理财/存款/债券/其他 |
| 买入日期 | 记录日期 |
| 买入金额 | 本次投入金额 |
| 买入单价 | 可选（股票输入股数/单价） |
| 持有份额 | 可选 |
| 账户 | 从哪个账户买入 |
| 备注 | 投资说明 |

#### 4.2.2 每月更新

| 字段 | 说明 |
|------|------|
| 更新月份 | 如：2026-04 |
| 当前市值 | 每月更新当前价值 |
| 盈亏金额 | 自动计算 vs 成本价 |
| 分红收益 | 可选，记录分红 |
| 备注 | 本月变化说明 |

#### 4.2.3 投资统计

- **持仓汇总**：总投入、总市值、累计收益、收益率
- **月报对比**：本月 vs 上月市值变化
- **资产配置**：各类投资占比饼图
- **收益曲线**：历史收益走势

### 4.3 投资 API 设计

```python
# 投资记录
POST   /investments              # 添加投资记录（买入）
GET    /investments              # 投资列表
GET    /investments/{id}         # 投资详情
PUT    /investments/{id}         # 更新投资信息
DELETE /investments/{id}         # 删除投资记录

# 市值更新
POST   /investments/{id}/values  # 记录月度市值
GET    /investments/{id}/values  # 市值历史

# 统计
GET    /investments/portfolio     # 投资组合汇总
GET    /investments/stats        # 投资收益统计
```

### 4.4 投资相关数据模型

```python
class Investment(Base):
    """投资记录表"""
    id: int
    user_id: int
    account_id: int              # 对应账户（证券/基金账户）
    name: str                    # 投资名称
    type: str                    # stock/fund/理财/deposit/bond/other
    purchase_date: date          # 买入日期
    purchase_amount: Decimal     # 买入金额
    purchase_price: Decimal      # 买入单价（可选）
    shares: Decimal               # 持有份额（可选）
    remark: str

class InvestmentValue(Base):
    """投资市值记录"""
    id: int
    investment_id: int
    update_month: str            # 2026-04
    current_value: Decimal        # 当前市值
    profit_loss: Decimal          # 盈亏金额
    dividend: Decimal            # 分红金额
    remark: str
    created_at: datetime
```

## 五、预算管理功能

### 5.1 预算设置

- **月度总预算**：设置每月总支出上限
- **分类预算**：针对重点分类设置单独预算
  - 孩子支出（可细分）
  - 家庭不固定支出
  - 餐饮专项
  - 服装专项
  - 其他自定义

### 5.2 预算提醒

| 触发条件 | 提醒内容 |
|----------|----------|
| 达到 80% | "本月 {{分类}} 已使用 80%，注意控制" |
| 达到 100% | "{{分类}} 预算已用完" |
| 超预算 | "{{分类}} 已超预算 ¥XXX" |

### 5.3 预算 API 设计

```python
POST   /budgets                  # 创建/更新月度预算
GET    /budgets                  # 获取预算列表
GET    /budgets/current          # 获取当月预算
GET    /budgets/summary          # 预算执行情况
PUT    /budgets/{id}             # 更新预算
DELETE /budgets/{id}             # 删除预算
```

## 六、家庭协作功能

### 6.1 家庭成员

| 角色 | 权限 |
|------|------|
| owner（创建者） | 家庭所有权限，可解散家庭 |
| admin（管理员） | 管理成员、查看所有账单 |
| member（成员） | 仅看自己的账单 |

### 6.2 成员功能

- 查看家庭成员列表及角色
- 邀请家庭成员（生成邀请码）
- 加入家庭（输入邀请码）
- 设置成员角色
- 移除成员

### 6.3 家庭账单

- 家庭成员共享所有账户和分类
- 账单记录归属到具体成员
- 家庭统计包含所有成员数据
- 支持按成员筛选账单

## 七、数据统计维度

### 7.1 支出统计

- **按时间**：日/周/月/年/自定义区间
- **按分类**：各分类支出金额及占比
- **按成员**：家庭成员支出对比
- **按账户**：各账户支出情况
- **环比分析**：与上期对比变化

### 7.2 收入统计

- **按时间**：月/年统计
- **按分类**：各收入类型占比
- **收入趋势**：历史收入走势

### 7.3 资产负债

- **资产**：现金、存款、投资市值
- **负债**：信用卡欠款、贷款等
- **净资产**：资产 - 负债
